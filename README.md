# 📧 Gmail Sent Email Extractor (Last 2 Months)

This Google Apps Script extracts all unique email addresses from your **sent emails** over the last 2 months and writes them into a Google Sheet.

---

## 🚀 Features

- Scans Gmail **Sent folder**
- Filters emails from the **last 2 months**
- Extracts recipients from:
  - To
  - CC
  - BCC
- Removes duplicate email addresses
- Outputs results directly to a Google Sheet

---

## 📂 Project Structure
├── Code.gs # Main Apps Script file
└── README.md # Documentation


---

## 🛠️ Setup Instructions

1. Open **Google Sheets**
2. Go to **Extensions → Apps Script**
3. Create a new project
4. Paste the script into `Code.gs`
5. Save the project

---

## ▶️ How to Run

1. In Apps Script, select the function:

getSentEmailAddressesLast2Months

2. Click **Run**
3. Authorize the script when prompted
4. Check your Google Sheet — results will appear automatically
